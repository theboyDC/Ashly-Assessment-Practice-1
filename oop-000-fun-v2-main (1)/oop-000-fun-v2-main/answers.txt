    A build tool is like a personal assistant that you communicate with through an .xml file, you write what you would like her to do and what to use while doing this.
    Build tools help with monitoring your project's lifecycle, and figuring out which part of it doesn't work.
    Maven brings the ability to check your project's lifecycle, The plugins you have used as specified on you .xml file, The dependencies that have been used also specified on your .xml file and lastly it helps you find out which repo you are working on if you are working in one.
    Although I prefer to use the built-in maven button on IntelliJ there are commands you have to install first(the button has a sync option which does this automatically) you'd first run mvn clean, mvn compile, mvn test there are more but these are the ones we'd use every day.
